Scalar Transport Guide

Numerical methods for diffusion, convection, and transient transport using FiPy and NumPy

Overview

This guide covers the numerical solution of scalar transport problems using FiPy, a finite-volume PDE solver for Python. It is intended as a practical reference for problems involving:

  • Stationary diffusion (Laplace equation)
  • Steady-state convection–diffusion
  • Transient diffusion and transport
  • Full scalar transport with source terms
Note FiPy uses the finite volume method (FVM). Unlike finite differences, FVM ensures local conservation on each control volume, making it well-suited for transport problems.

The general scalar transport equation is:

∂φ/∂t + ∇·(u φ) = ∇·(D ∇φ) + S

Special cases — Laplace, diffusion, convection–diffusion — emerge by setting appropriate terms to zero.

General Workflow

Every FiPy simulation follows the same five-step structure, regardless of the physical problem:

StepActionFiPy construct
1Define the domainGrid1D, Grid2D, Grid3D
2Define the unknown fieldCellVariable
3Apply boundary conditions.constrain(), .faceGrad.constrain()
4Write the PDEDiffusionTerm, TransientTerm, etc.
5Solve.solve() or .sweep()

Mesh Definition

The mesh discretises the physical domain into control volumes. FiPy provides uniform structured meshes for 1D, 2D, and 3D problems.

Grid1D(dx, nx)
1D mesh of nx cells, each of width dx.
Grid2D(dx, dy, nx, ny)
2D rectangular mesh.
Grid3D(dx, dy, dz, nx, ny, nz)
3D block mesh.

Example — 1D mesh

L, N = 1.0, 100
dx   = L / N
mesh = Grid1D(dx=dx, nx=N)

# Cell-center coordinates (useful for plotting)
x = np.array(mesh.cellCenters[0])

Example — 2D mesh

Lx, Ly = 1.0, 0.4
Nx, Ny = 150, 60
dx, dy = Lx/Nx, Ly/Ny

mesh = Grid2D(dx=dx, dy=dy, nx=Nx, ny=Ny)

# For visualisation, reshape cell data to (Ny, Nx)
# FiPy stores data in Fortran order:
phi_2d = np.array(phi.value).reshape((Nx, Ny), order="F").T
Important — Array reshaping FiPy stores cell values in Fortran (column-major) order. Always reshape with order="F" and transpose to obtain a (Ny, Nx) array compatible with matplotlib's contourf.

Variables

CellVariable holds the unknown field at every cell centre. It supports arithmetic, gradients, and constraints.

name
Label for the variable (used in output).
mesh
The mesh the variable lives on.
value
Initial value — scalar (uniform) or array (non-uniform).

Uniform initial condition

phi = CellVariable(name="concentration", mesh=mesh, value=0.0)

Non-uniform initial condition

x       = np.array(mesh.cellCenters[0])
c_init  = np.where(x < 0.10, 1.0, 0.0)   # step profile
c       = CellVariable(name="c", mesh=mesh, value=c_init)

Boundary Conditions

Dirichlet — fixed value

Use .constrain(value, where) to pin the variable on a set of faces.

phi.constrain(1.0, mesh.facesLeft)    # φ(0) = 1
phi.constrain(0.0, mesh.facesRight)   # φ(L) = 0

Neumann — fixed flux

Use .faceGrad.constrain([value], where) to fix the normal gradient.

# Homogeneous Neumann (insulated / no-flux) — also the FiPy default
phi.faceGrad.constrain([0.0], mesh.facesTop)

# Non-homogeneous Neumann (prescribed flux)
phi.faceGrad.constrain([-0.4], mesh.facesRight)
Note FiPy imposes a homogeneous Neumann condition by default on any face that has no explicit constraint. You only need to call .faceGrad.constrain([0.0], ...) when you want to be explicit or when other faces do carry a flux.
Face selectorMeaning
mesh.facesLeftx = 0
mesh.facesRightx = Lx
mesh.facesBottomy = 0
mesh.facesTopy = Ly

Equations & Terms

Physical termPDE termFiPy class
Time derivative∂φ/∂tTransientTerm()
Diffusion∇·(D ∇φ)DiffusionTerm(coeff=D)
Convection∇·(u φ)ExponentialConvectionTerm(coeff=[[u]])
Linear source/sink−λ φImplicitSourceTerm(coeff=λ)
Uniform sourceSadd S to the right-hand side

Common equations

Laplace (stationary, no source)

eq = DiffusionTerm(coeff=D)        # ∇·(D ∇φ) = 0

Transient diffusion

eq = TransientTerm() == DiffusionTerm(coeff=D)

Steady convection–diffusion

eq = DiffusionTerm(coeff=D) - ExponentialConvectionTerm(coeff=[[u]])

Full transient transport with source and sink

eq = (TransientTerm()
     + ExponentialConvectionTerm(coeff=[[u]])
     == DiffusionTerm(coeff=D) - ImplicitSourceTerm(coeff=lam) + S)
Tip — ExponentialConvectionTerm Always use ExponentialConvectionTerm (not ConvectionTerm) for convection–diffusion problems. It uses an exponential interpolation scheme that is stable for all Péclet numbers and avoids spurious oscillations.

Solving: solve() vs sweep()

solve()sweep()
Iterations per callUntil convergenceOne
Residual accessNoYes — return value
Convergence controlInternalManual
Best forQuick prototypesDiagnostics, large systems

Using solve()

eq.solve(var=phi)     # stationary problems
eq.solve(var=phi, dt=dt)  # transient step

Using sweep() with convergence monitoring

tol, max_iter = 1e-8, 5000
residuos = []

for i in range(max_iter):
    res = eq.sweep(var=phi)
    residuos.append(res)
    if res < tol:
        print(f"Converged in {i+1} iterations")
        break

# Transient: one sweep per time step
res = eq.sweep(var=phi, dt=dt)

Péclet Number

The Péclet number quantifies the relative importance of convection versus diffusion:

Pe = u L / D
RegimeCriterionProfile shape
Diffusion-dominatedPe ≪ 1Linear
Moderate convectionPe ~ 1–10Curved
Strong convectionPe ≫ 1Flat with boundary layer
Pe = (u * L) / D

if   Pe < 1:   regime = "diffusion-dominated"
elif Pe <= 10: regime = "moderate convection"
else:          regime = "strong convection"

print(f"Pe = {Pe:.2f} → {regime}")

Example 1 — Laplace 1D: Chemical Potential in a Membrane

Steady-state transport of chemical potential μ(x) across a 1D membrane. No sources, no convection — pure diffusion at equilibrium.

d²μ/dx² = 0    with    μ(0) = μ_A,   μ(L) = μ_B

Analytical solution: μ(x) = μ_A + (μ_B − μ_A) · x / L (linear).

from fipy import CellVariable, Grid1D, DiffusionTerm

L, N   = 1.0, 101
dx     = L / (N - 1)
mu_A, mu_B = 1.0, 0.2
omega  = 1.8
tol    = 1e-8

# FiPy solution
mesh = Grid1D(dx=dx, nx=N-1)
mu   = CellVariable(name="mu", mesh=mesh, value=0.0)
mu.constrain(mu_A, mesh.facesLeft)
mu.constrain(mu_B, mesh.facesRight)

eq = DiffusionTerm(coeff=1.0)
res, it = 1e10, 0
while res > tol:
    res = eq.sweep(var=mu)
    it += 1

print(f"Converged: {it} iterations, residual = {res:.3e}")

# Analytical solution
x_fp   = np.array(mesh.cellCenters[0])
mu_ana = mu_A + (mu_B - mu_A) / L * x_fp
error  = np.max(np.abs(np.array(mu.value) - mu_ana))
print(f"Max error vs analytical: {error:.3e}")

Example 2 — Laplace 2D: Heat Conduction in a Plate

Temperature distribution T(x,y) in a rectangular plate with fixed temperatures on left/right borders and insulated top/bottom.

∇²T = 0
from fipy import CellVariable, Grid2D, DiffusionTerm

Lx, Ly = 1.0, 0.4
Nx, Ny = 150, 60
dx, dy = Lx/Nx, Ly/Ny
Th, Tc, k = 145.0, 85.0, 385.0

mesh = Grid2D(dx=dx, dy=dy, nx=Nx, ny=Ny)
T    = CellVariable(name="Temperature", mesh=mesh, value=0.0)

T.constrain(Th, mesh.facesLeft)
T.constrain(Tc, mesh.facesRight)
# Top/bottom: default homogeneous Neumann (insulated)

eq  = DiffusionTerm(coeff=k)
res, it = 1e10, 0
while res > 1e-6 and it < 4000:
    res = eq.sweep(var=T)
    it += 1

# Reshape for plotting
T_grid = np.array(T.value).reshape((Nx, Ny), order="F").T
x_c = np.linspace(dx/2, Lx-dx/2, Nx)
y_c = np.linspace(dy/2, Ly-dy/2, Ny)
X, Y = np.meshgrid(x_c, y_c)

# Heat flux: q = -k ∇T
dTdx, dTdy = np.gradient(T_grid, dx, dy)
qx, qy = -k * dTdx, -k * dTdy

plt.figure(figsize=(12, 4))
plt.subplot(1,2,1)
plt.contourf(X, Y, T_grid, 50, cmap="BuPu")
plt.colorbar(label="T (°C)")
plt.title("Temperature distribution")
plt.subplot(1,2,2)
plt.streamplot(X, Y, qx, qy, color=np.sqrt(qx**2+qy**2), cmap="BuPu")
plt.title("Heat flux lines")
plt.tight_layout()
plt.show()

Example 3 — Laplace 3D: Vacancy Concentration in a Crystal

Three-dimensional Laplace problem on a cubic domain. Demonstrates Grid3D and 3D data extraction for a mid-plane slice.

from fipy import CellVariable, Grid3D, DiffusionTerm

Lx=Ly=Lz=1.0;  Nx=Ny=Nz=20
dx=Lx/Nx; dy=Ly/Ny; dz=Lz/Nz

mesh = Grid3D(dx=dx, dy=dy, dz=dz, nx=Nx, ny=Ny, nz=Nz)
c    = CellVariable(name="vacancy concentration", mesh=mesh, value=0.0)
c.constrain(1.0, mesh.facesLeft)
c.constrain(0.2, mesh.facesRight)

eq = DiffusionTerm(coeff=1.0)
for i in range(2000):
    res = eq.sweep(var=c)
    if res < 1e-8: break

# Reshape to 3D array c_3d[ix, iy, iz]
c_3d    = np.array(c.value).reshape((Nx,Ny,Nz), order="F")
iz_mid  = Nz // 2
c_slice = c_3d[:, :, iz_mid].T    # (Ny, Nx)

Example 4 — Steady Convection–Diffusion 1D

Contaminant transport in a channel. The Péclet number governs the shape of the concentration profile.

D d²φ/dx² − u dφ/dx = 0

Analytical solution: φ(x) = (1 − e−(u/D)x) / (1 − e−(u/D)L)

from fipy import CellVariable, Grid1D, DiffusionTerm, ExponentialConvectionTerm

L, N   = 10.0, 400
u, D   = 10.0, 15.0
dx     = L / N
Pe     = u * L / D
print(f"Péclet number: Pe = {Pe:.2f}")

mesh = Grid1D(dx=dx, nx=N)
phi  = CellVariable(name="concentration", mesh=mesh, value=0.0)
phi.constrain(0.0, mesh.facesLeft)
phi.constrain(1.0, mesh.facesRight)

eq = DiffusionTerm(coeff=D) - ExponentialConvectionTerm(coeff=[[-u]])
eq.solve(var=phi)

x_num   = np.array(mesh.cellCenters[0])
phi_ana = (1 - np.exp(-u/D * x_num)) / (1 - np.exp(-u/D * L))
error   = np.max(np.abs(np.array(phi.value) - phi_ana))
print(f"Max error: {error:.3e}")

Example 5 — Transient Diffusion

Time-dependent diffusion of dopants in a semiconductor wafer. Profiles are saved at selected times.

∂c/∂t = D ∂²c/∂x²
from fipy import CellVariable, Grid1D, TransientTerm, DiffusionTerm

L, N  = 5e-6, 100
D, dx = 2e-14, L/N

mesh = Grid1D(dx=dx, nx=N)
c    = CellVariable(name="dopant", mesh=mesh, value=0.0)
c.constrain(1.0, mesh.facesLeft)   # surface concentration
# Right: insulated by default

eq = TransientTerm() == DiffusionTerm(coeff=D)

dt          = 1000.0
t_final     = 5e6
tiempos_obj = [1e5, 5e5, 1e6, 2e6, 5e6]
soluciones  = {}
t = 0.0

while t <= t_final:
    eq.sweep(var=c, dt=dt)
    t += dt
    for to in tiempos_obj:
        if abs(t - to) < dt/2:
            soluciones[to] = np.array(c.value)

Example 6 — Full Transport: Convection + Diffusion + Sink

Gas tracer in a calibration chamber with flow, diffusion, and wall absorption.

∂c/∂t + u ∂c/∂x = D ∂²c/∂x² − λ c
from fipy import (CellVariable, Grid1D, TransientTerm,
                  DiffusionTerm, ExponentialConvectionTerm, ImplicitSourceTerm)

L, N   = 0.50, 150
u      = 1e-3
D      = 2e-4
lam    = 5e-3
beta   = -0.40
dx     = L / N

mesh   = Grid1D(dx=dx, nx=N)
x      = np.array(mesh.cellCenters[0])
c_init = np.where(x < 0.10, 1.0, 0.0)
c      = CellVariable(name="tracer", mesh=mesh, value=c_init)

c.constrain(0.0, mesh.facesLeft)
c.faceGrad.constrain([beta], mesh.facesRight)

eq = (TransientTerm()
     + ExponentialConvectionTerm(coeff=[[u]])
     == DiffusionTerm(coeff=D) - ImplicitSourceTerm(coeff=lam))

dt = 10.0
while t <= 1000.0:
    eq.sweep(var=c, dt=dt)
    t += dt

Convergence & Residuals

sweep() returns the residual norm after each iteration. Plotting it on a log scale reveals convergence behaviour.

plt.figure(figsize=(8, 4))
plt.semilogy(residuos, color='blueviolet', lw=2)
plt.axhline(tol, color='lightgreen', ls='--', lw=1.5, label=f'tol = {tol:.0e}')
plt.xlabel('Iteration')
plt.ylabel('Residual (log scale)')
plt.title('Convergence of iterative solver')
plt.legend()
plt.grid(True, which='both', alpha=0.4)
plt.tight_layout()
plt.show()

Error Metrics

When an analytical solution exists, quantify numerical accuracy with three standard metrics:

Maximum error
Worst-case pointwise deviation.
Mean absolute error (MAE)
Average pointwise deviation.
Root mean square error (RMSE)
Penalises large deviations more heavily.
phi_num = np.array(phi.value)
phi_ana = ...   # analytical expression evaluated at x_num

error_abs  = np.abs(phi_num - phi_ana)
error_max  = np.max(error_abs)
error_mae  = np.mean(error_abs)
error_rmse = np.sqrt(np.mean(error_abs**2))

print(f"Max error:  {error_max:.3e}")
print(f"MAE:        {error_mae:.3e}")
print(f"RMSE:       {error_rmse:.3e}")

Execution Time

import time

inicio = time.perf_counter()

# ... solve / sweep loop ...

fin    = time.perf_counter()
print(f"Execution time: {fin - inicio:.4f} s")
Note Use time.perf_counter() rather than time.time() for short intervals — it has higher resolution and is not affected by system clock adjustments.
